# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added
- Initial release
- Multi-size image processing (thumb, medium, large)
- Clean architecture (Controller → Service → Repository → Trait)
- Tailwind CSS UI (no Bootstrap dependency)
- Folder organization system
- WebP conversion support
- Config-driven behavior
- Type-safe code with full type hints
- Laravel 12+ and PHP 8.3+ support
- Image upload with validation
- Media metadata management (title, alt, caption, description)
- Search functionality
- RESTful API endpoints
- Automatic file cleanup on deletion

### Features
- Image resizing with aspect ratio preservation
- Multiple image size generation
- Storage disk configuration
- Quality control settings
- File size limits
- MIME type validation
- Folder hierarchy support
- Media search and filtering
